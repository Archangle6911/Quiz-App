📘 QuizMaster — Interactive Quiz App UI

A sleek and modern quiz application interface designed in Figma and implemented using pure HTML + CSS.
This project focuses on visual design, clean layout structure, responsiveness, and future expansion with JavaScript.

🚀 Features

✔️ Elegant hero section with title & subtitle
✔️ Soft grid background and neon-style blobs
✔️ Responsive layout (desktop, tablet, mobile)
✔️ Minimalist top navigation (About / Login)
✔️ Smooth typography using Playfair Display + Inter
✔️ Interactive icons with soft hover effects
✔️ Clean folder structure & ready for JavaScript integration
